# 平安银行账单分析系统 - 跨平台应用

## 📱 快速启动

### Mac 用户

**方式一：双击应用程序**
```
双击 "平安银行账单分析.app" 即可启动
```

**方式二：使用启动脚本**
```
双击 "启动应用.command" 启动
```

### Windows 用户

**方式一：创建快捷方式（推荐）**
```
1. 双击运行 "create_windows_shortcut.bat"
2. 双击生成的 "平安银行账单分析.lnk" 启动
```

**方式二：使用启动脚本**
```
双击 "启动应用.bat" 启动
```

## 🎨 创建应用程序图标

如果您想自定义应用图标，可以运行：
```
Mac/Linux: ./创建应用.command
Windows:   双击运行 create_windows_shortcut.bat
```

## 📂 目录结构

```
pingan-dashboard/
├── 平安银行账单分析.app      # Mac应用程序包
├── 平安银行账单分析.lnk      # Windows快捷方式（需创建）
├── 启动应用.command          # Mac启动脚本
├── 启动应用.bat              # Windows启动脚本
├── create_mac_app.sh         # Mac应用创建脚本
├── create_windows_shortcut.bat # Windows快捷方式创建脚本
├── 创建应用.command          # 一键创建应用脚本
├── icon.png                  # 应用图标PNG
├── icon.icns                 # Mac图标文件
├── icon.ico                  # Windows图标文件
├── desktop_app.py            # 桌面应用主程序
├── frontend/                 # 前端界面
└── backend/                  # 后端服务
```

## 🔧 功能特性

- ✅ 黑暗模式（默认）
- ✅ 月度消费热力图
- ✅ 账单明细按年份分组
- ✅ 金额从大到小排序
- ✅ 跨平台支持（Mac/Windows）

## 📧 使用说明

1. 启动应用后，输入邮箱地址和授权码
2. 选择邮箱类型（163/QQ/Gmail/阿里云）
3. 点击"开始读取"按钮
4. 等待应用读取并解析账单
5. 查看消费分析和账单明细

## 🎯 系统要求

- **Mac**: macOS 10.13 或更高版本
- **Windows**: Windows 7 或更高版本
- **Python**: Python 3.7 或更高版本
