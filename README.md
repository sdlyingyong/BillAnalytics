# 平安银行账单分析系统

一个智能的银行账单分析工具，自动从邮箱读取平安银行信用卡账单，提供详细的消费分析和可视化报告。

## 功能特性

### 📊 数据分析
- **月度消费趋势** - 查看每月消费变化
- **消费类别分布** - 了解消费结构
- **消费对比分析** - 本月vs上月环比分析
- **消费热力图** - 一周消费分布
- **消费排行** - Top商家统计

### 🔍 智能功能
- **智能分类** - 自动识别9大消费类别
- **智能过滤** - 自动过滤还款、退款、费用记录
- **多维度搜索** - 按商家、金额、日期搜索
- **灵活分组** - 按日期、月份、年份、分类分组

### 📈 数据导出
- **Excel导出** - 完整的账单数据表格
- **PDF导出** - 格式化的分析报告

### 🎨 用户体验
- **深色主题** - 护眼的暗色界面
- **响应式设计** - 支持各种屏幕尺寸
- **实时筛选** - 即时查看筛选结果

## 技术栈

### 后端
- Python 3.x
- Flask - Web框架
- PyPDF2 - PDF解析
- POP3/IMAP - 邮件协议

### 前端
- HTML5/CSS3/JavaScript
- Chart.js - 数据可视化
- SheetJS (xlsx) - Excel导出
- jsPDF - PDF导出
- Axios - HTTP客户端

## 安装

### 方式一：本地运行

#### 1. 克隆仓库
```bash
git clone https://github.com/sdlyingyong/BillAnalytics.git
cd BillAnalytics
```

#### 2. 安装Python依赖
```bash
pip install -r backend/requirements.txt
```

#### 3. 启动后端服务
```bash
cd backend
python app.py
```

后端服务将运行在 http://localhost:8080

#### 4. 启动前端服务
```bash
cd frontend
python -m http.server 8081
```

前端页面访问地址：http://localhost:8081/index.html

### 方式二：部署到Vercel（推荐）

#### 1. 安装Vercel CLI
```bash
npm install -g vercel
```

#### 2. 登录Vercel
```bash
vercel login
```

#### 3. 部署项目
```bash
cd BillAnalytics
vercel
```

#### 4. 按照提示操作
- 选择您的账户
- 确认项目名称
- 等待部署完成

部署成功后，Vercel会提供一个在线访问地址，例如：`https://bill-analytics.vercel.app`

**注意：** Vercel免费版有一些限制：
- 函数执行时间限制：10秒
- 内存限制：1024MB
- 不支持持久化存储

## 使用说明

### 1. 获取邮箱授权码

#### QQ邮箱
1. 登录QQ邮箱网页版
2. 设置 -> 账户 -> POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务
3. 开启POP3/SMTP服务
4. 生成授权码

#### 163邮箱
1. 登录163邮箱
2. 设置 -> POP3/SMTP/IMAP
3. 开启POP3/SMTP服务
4. 设置授权码

### 2. 使用系统
1. 选择邮箱类型（QQ/163/Gmail等）
2. 输入邮箱地址
3. 输入授权码（不是邮箱密码）
4. 点击"开始读取"

系统将自动：
- 连接邮箱
- 查找平安银行账单邮件
- 解析PDF附件
- 提取交易记录
- 生成分析报告

## 常见问题

### Q: 为什么连接邮箱失败？
A: 请确保：
1. 使用的是授权码，不是邮箱密码
2. 已开启POP3/SMTP服务
3. 邮箱类型选择正确

### Q: 为什么没有找到账单？
A: 请确保：
1. 邮箱中有平安银行的账单邮件
2. 账单邮件主题包含"平安银行"或"平安信用卡"
3. 账单是PDF格式附件

### Q: 数据会保存吗？
A: 数据仅在浏览器会话中保存，刷新页面后需要重新读取。
