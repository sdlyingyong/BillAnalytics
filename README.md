# 平安银行账单分析系统

智能银行账单分析工具，自动从邮箱读取平安银行信用卡账单，提供详细的消费分析和可视化报告。

## 🚀 快速启动

### Mac用户
```
双击 "平安银行账单分析.app" 启动应用
```

### Windows用户
```
1. 双击运行 "create_windows_shortcut.bat"
2. 双击 "平安银行账单分析.lnk" 启动应用
```

**首次运行会自动安装依赖，请耐心等待约1-2分钟。**

## 📱 使用方法

### 1. 获取邮箱授权码

**QQ邮箱**
1. 登录QQ邮箱网页版
2. 设置 → 账户 → POP3/SMTP服务
3. 开启服务并生成授权码

**163邮箱**
1. 登录163邮箱
2. 设置 → POP3/SMTP/IMAP
3. 开启服务并设置授权码

### 2. 启动应用

**Mac**: 双击 `平安银行账单分析.app`

**Windows**: 运行 `create_windows_shortcut.bat` 后双击快捷方式

### 3. 输入信息
1. 选择邮箱类型（163/QQ/Gmail/阿里云）
2. 输入邮箱地址
3. 输入授权码（不是邮箱密码）
4. 点击"开始读取"

系统将自动读取并分析账单。

## ❓ 常见问题

**Q: 为什么连接邮箱失败？**
- 确保使用授权码，不是邮箱密码
- 确保已开启POP3/SMTP服务
- 确保邮箱类型选择正确

**Q: 为什么没有找到账单？**
- 确保邮箱中有平安银行账单邮件
- 确保账单是PDF格式附件

**Q: 数据会保存吗？**
- 数据仅在浏览器会话中保存
- 刷新页面后需要重新读取

## 🔒 安全说明

- 不要在代码中硬编码邮箱密码或授权码
- 授权码应妥善保管，不要分享给他人
- 建议定期更换邮箱授权码
